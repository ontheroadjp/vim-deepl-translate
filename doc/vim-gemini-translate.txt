INTRODUCTION                                *vim-deepl-translate-introduction*

This plugin provides a function to translate selected text into English using
the DeepL API.

==============================================================================
USAGE                                           *vim-deepl-translate-usage*

In line-wise visual mode (V), select the text you want to translate and press
`<leader>tr`. The selected text will be replaced with the English translation.

To use a different mapping, you can map your preferred key to
`<Plug>(vimgeminitranslate-translate)`.

Example: >
    xnoremap <leader>t <Plug>(vimgeminitranslate-translate)
<
==============================================================================
CONFIGURATION                            *vim-deepl-translate-configuration*

g:vimgeminitranslate_deepl_api_key~
  Specifies your DeepL API key. This is required.

  Example: >
    let g:vimgeminitranslate_deepl_api_key = 'YOUR_DEEPL_API_KEY'
